# 🏥 Sistema Hospitalar

Um sistema web para gerenciamento hospitalar desenvolvido com Django, com um foco para diminuir filas para marcar consultas e exames sem ter muitos problemas, poder ter resultados dos exames e receitas em pdf enviada pelos os medicos 

> **⚠️ Projeto em desenvolvimento**
>
> Este projeto ainda está sendo desenvolvido. Novas funcionalidades, melhorias de interface, otimizações e integrações serão adicionadas ao longo do desenvolvimento, Além de ter uma interface para os pacientes, terá também para os medicos para enviar os resultados e outras funcionalidades também

---

## ✨ Funcionalidades atuais

* Cadastro de pacientes
* Login com autenticação
* Visualização do perfil do paciente
* Agendamento de consultas e exames
* Listagem de exames
* Visualização de resultados de exames
* API REST utilizando Django REST Framework
* Autenticação com JWT
* Banco de dados MySQL
